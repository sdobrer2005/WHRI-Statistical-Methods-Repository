# Statistical Methods Repository

## **Overview**

This repository was developed as a **teaching and reference resource** to support applied statistical work within the **Women’s Health Research Institute (WHRI)**. It provides structured, reproducible implementations of commonly used statistical methods, with parallel code in **SAS and R**.

The primary objective is to facilitate **consistent analytical practices**, improve **transparency**, and support learning through **practical, working examples**. Each method is accompanied by clearly documented code and concise methodological explanations.

---

## **Scope**

The repository includes:

- Implementations of standard statistical methods in **SAS and R**
- Side-by-side comparisons across software environments
- Reproducible examples using publicly available or simulated data
- Supporting documentation describing **assumptions**, **interpretation**, and **appropriate use**

The focus is on methods commonly used in health research, with an emphasis on **clarity**, **reproducibility**, and **practical application**.

---

## **Initial Content**

The first module includes:

### **Two-Sample t-Test**

- Independent samples comparison  
- Assumption checking (including **normality assessment**)  
- Implementation in **SAS and R**  
- Interpretation of results  

This serves as a foundational example for understanding statistical workflows, including **data preparation**, **assumption evaluation**, **model execution**, and **reporting**.

---

## **Intended Use**

This repository is designed to:

- Support analysts and trainees in developing **applied statistical skills**  
- Provide reusable code templates for common analyses  
- Promote **standardized approaches** to statistical methodology  

While developed as a WHRI teaching resource, the materials are publicly available and may be adapted for broader use.
